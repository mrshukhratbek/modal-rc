<h1 align="center">Modal react</h1>

<div align="center">
Modal component for React.js
</div>

## Installation

```sh
// with npm
npm install modal-rc

// with yarn
yarn add modal-rc
```

## Usage

```jsx
import * as React from "react";
import Modal from "modal-rc";

function App() {
  return <Modal>Content</Modal>;
}
```
